Notes for iPhone sound

To use as a home-screen app, open the HTML in Safari, then Share → Add to Home Screen.

iPhone audio needs a real tap before any sound can play — tapping Start (or any button) unlocks it for the session. If sound still doesn't play, check these iPhone-level settings, which no web app can override:

• The physical Ring/Silent switch on the side of the phone — if it shows orange, all web sounds are muted.
• Low Power Mode and Focus modes can also mute or interrupt background audio.
• If the screen locks, iOS suspends the sound engine; this app requests a screen wake lock while a session is running to help keep it awake, but you may still need to disable Auto-Lock in Settings → Display & Brightness for long sessions.

As a backup, the timer screen flashes on every cue in addition to the sound, so you still get a signal if the phone is on silent.
